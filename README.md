# 招聘：ChatGPT 模型执行助理 / AI 数据生产助理

我正在做两个方向的 AI 模型：

## 一、故事生产模型

类似好莱坞故事流水线工厂。

用模型拆解故事结构、角色关系、场景推进、冲突设计、节奏变化，并批量生产可复用的故事样本和案例数据。

## 二、多语言学习模型

面向英语、西班牙语、德语等语言学习。

不是传统背单词、背语法路线，而是通过场景、表达、语境、结构，生产更接近真实使用场景的语言学习数据。

---

目前模型的 **0-1 已经完成**。

现在需要找人协助完成 **1-100 的执行工作**。

这不是研究岗。
不是编程岗。
不是自由创作岗。
也不是机器学习算法岗。

这个岗位的核心是：

**基于已有模型，用 ChatGPT 反复生产样本、清洗数据、整理案例、优化流程，把模型从 1 推进到 100。**

工作会比较枯燥，需要耐心、细心和稳定执行力。

---

# 主要工作内容

1. 按照已有模型说明，用 ChatGPT 生成故事 / 多语言相关样本。
2. 对生成结果进行筛选、分类、清洗、修正。
3. 把零散内容整理成结构化数据。
4. 记录哪些 prompt / instructions 有效，哪些无效。
5. 按规则整理故事样本、多语言样本、案例样本。
6. 发现模型执行中的问题，并反馈优化意见。
7. 协助把模型流程整理成可重复使用的操作文档。
8. 长期做重复性的 AI 数据生产和模型执行工作。

---

# 岗位要求

## 1. 会用 ChatGPT

不要求专家级，但要真正用过 ChatGPT，能理解 prompt、instructions、上下文、改写、批量生成等基本操作。

## 2. 逻辑能力强

能看出内容是否跑偏，能判断结构是否一致，能发现生成结果里的问题。

## 3. 有耐心

这个工作不是每天都有新鲜感。
很多时候就是反复生成、反复检查、反复修改、反复整理。

## 4. 执行稳定

能持续交付，不失联，不拖太久。

## 5. 文字理解能力好

能理解复杂说明，能按规则执行，而不是凭感觉乱改。

## 6. 尽量通过 Telegram 沟通

日常沟通希望主要使用 TG。

# 测试任务

正式合作前，会先做一个小测试：

1. 给你一段模型说明。
2. 给你一组样本。
3. 让你用 ChatGPT 按规则生成 10-20 条数据。
4. 再让你筛选、清洗、标注、整理。
5. 看你是否能稳定理解规则并执行。

合适的话，可以长期合作。
